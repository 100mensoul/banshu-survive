## 実装サマリー

### 土台（前段階で確立・今回も維持）

| 項目 | 内容 |
|------|------|
| データモデル | `source` / `target` を `Endpoint`（`kind:'node'` \| `kind:'edge'`）+ `waypoints[]` |
| 単一ルータ | `computePath()` のみ（stub 24px → 直交化） |
| 描画 | 矢印はポリゴン1本。ドラッグ用ハンドルは透明ヒットエリア |
| カード外枠 | DOM 計測 `boundsFromDomRect` で矢印先端を白枠の辺上に |
| 自動コーナー | 見える角はすべてドラッグ可（昇格して waypoint 化） |
| 夫婦線接続 | 子の `target` を `kind:'edge'`（夫婦線上の弧長 `t`）に。2パス描画 |

### 今回追加した自動配線（A〜D）

| 記号 | 機能 | 概要 |
|------|------|------|
| **A** | ポート自動分散 | 同じ (ノード, 辺) への自動接続を `t=(i+1)/(n+1)` で均等配置。手動端点は対象外 |
| **B** | 兄弟トランク | 同一夫婦線に子2人以上 → 共有トランクX + 横バス（`TRUNK_BUS_OFFSET=40px`） |
| **C** | 軽い衝突回避 | 他カードを障害物として、直結が貫通する場合は上/下レーンへ迂回 |
| **D** | 相対 waypoint | ドラッグ確定時 `{seg,u}` に変換。旧 `{x,y}` も読み込み互換 |

### 保守・性能（今回）

| 項目 | 内容 |
|------|------|
| 死にコード削除 | 旧ルーター約560行削除（`orthogonalRouteBetweenBounds` 系など） |
| ドラッグ最適化 | `requestAnimationFrame` で描画を1フレーム1回に集約 + `getRenderPlan` キャッシュ |
| 斜め配置バグ修正 | `defaultPortsBetweenBounds` の上側 `t` を左右で `0.62` / `0.38` に分散 |
