## 状態（現行・2026-05-15 追記）

**ナビのカクつき対策として、デスクトップとモバイルで別手段を採用した。**

### デスクトップ（`min-width: 901px`）

- **`index.html` の DOM 順**を **`ヒーロー → 石垣ナビ → 真実のストーリー`** に変更（`home-top-stack` でラップ）。本文アニメで帯の高さが変わっても、ナビは常にヒーロー直下のため **縦位置は動かない**。
- ゴーストレイヤーは **使わない**（`truth-story__ghost` は CSS で `display: none`）。

### モバイル（`max-width: 900px`）

- **見た目の順**は従来どおり **ヒーロー → 真実のストーリー → ナビ** にしたいため、同じ DOM に対し **`.home-top-stack` の子に `flex` の `order`**（1: ヒーロー、2: 本文、3: ナビ）を指定。
- **ゴースト + ライブ**（非表示の全文で高さ確保、`truth-story__live` を `absolute` で重ねる）を **モバイルのメディアクエリ内のみ**有効化。ナビの `order: 3` のため、本文の伸びでナビが押し下がらない想定。

### 注意

- **モバイルでは DOM 順と見た目順が異なる**ため、キーボードのフォーカス順は **ヒーロー → ナビ → 本文**（DOM 順）になる。視覚順と一致しない点は許容とする（必要なら別途 `tabindex` 等で検討）。
- **`prefers-reduced-motion: reduce`**: `fillStatic()` が `section` に **`.truth-story--reduced`** を付与し、ゴーストを無効化して通常フローに戻す。

**キャッシュ:** `main.css?v=navmichi47`、`truth-story-reveal.js?v=4`。

**ゴースト文面の保守:** `index.html` の `.truth-story__ghost` と `truth-story-reveal.js` の `DATA` / `fillStatic()` を常に一致させる。
