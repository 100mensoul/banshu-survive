```javascript
function activeLayerId() {
  if (isKonuiZoned() && currentZoneId) return `${LAYER_ID}-${currentZoneId}`;
  if (isKonuiZoned() && konuiOverviewMode) return `${LAYER_ID}-overview`;
  return LAYER_ID;  // 常に 'hirao' ベース（world_id とは別）
}
```

const layerIds = [activeLayerId()];

  layerIds.push(LAYER_ID);  // 'hirao' を第二候補
}

## F. 高架パイプライン（疑似コード）

`rebuildHighwayViaduct()` の流れ。詳細は L5 区間を参照。

```
if viewMode !== '3d' → 非表示
for each cell i:
  if !elevatedHighwayCell(i) → continue
  theta = smoothedHighwayAngle(gx, gz)
  if !isHighwayCenterline(...) → continue  // 路面も中心線のみ（v81〜）
  deckWidth = metersToUnits(roadWidthMeters) + cell * 0.25
  alongU = cell * 2.35
  appendHighwayDeckPlanks(..., deckWidth, alongU)  // スラブ＋板＋溝
  girderSites.push(...)  // 28m間隔

mergeUnitBoxes(deckBoxes) → MeshStandardMaterial(highwayColor)
mergeUnitBoxes(grooveBoxes) → 暗い溝
mergeUnitBoxes(girderBoxes) → rotY=theta, sx=width*0.5, sz=4m
```

**既知の限界:** グリッド地形＋ボックス合成のため、斜め道路の外縁に段差（ギザギザ）が残りやすい。
