# js/world/ — 白鷺の世界 新機能モジュール置き場

マスタープラン: `docs/世界地図_マスタープラン_白鷺の世界.md`

## このフォルダの原則

1. **実寸メートル定義** — すべての物体・距離は実寸（メートル）で定義し、配置時に `metersToUnits()` で変換する。誇張しない。
2. **InstancedMesh** — 同種のプロップは種別ごとに `THREE.InstancedMesh` へ集約する（1種別 = 1〜3 draw call）。
3. **播州固有データはコードに書かない** — 色・寸法・地名などはデータ（Supabase行・設定オブジェクト）に置く。エンジンとワールドの分離。
4. **タッチファースト** — 入力は Pointer Events で書く（マウス・指・Apple Pencil を1系統で扱う）。iPad が主編集デバイス。
5. **既存コードを壊さない** — `world-map-editor.js` への変更は import 追加と呼び出し追加のみ。5,600行のリファクタはしない。
