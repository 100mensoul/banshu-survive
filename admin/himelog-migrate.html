ヒメログ移行ツール（旧Firebase → Supabase）

旧ヒメログ（Firebase Firestore の diaryEntries）を読み込み、選んだものを新ヒメログ（Supabase himelog_entries）へ保存します。 元の日付は保持され、同じ投稿を二度保存しないよう Firebase の元ID で重複チェックします。移行が終わったら、このファイルは削除して構いません。

Supabase 接続設定がありません。js/supabase-public-config.js を確認してください。
事前に Supabase で admin/sql/himelog.sql（または alter table … add column if not exists firebase_id text;）を実行し、firebase_id 列があることを確認してください。
